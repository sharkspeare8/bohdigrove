# 菩提树 · Bodhi Grove

> **A Digital Garden for Scholars – 滋养心灵的数字园林**

## 🌿 项目简介

菩提树是一个专为学生打造的东方美学数字社区，旨在数字时代的喧嚣中重建一份书院精神，创造一个宁静、优雅、富有文化底蕴的在线学习与交流空间。

**核心哲学**：取自禅宗智慧“菩提本无树，明镜亦非台”，寓意教育如明镜拂尘，智慧如菩提生长。

## ✨ 核心特色

### 🏮 东方美学设计
- **园林式界面**：借鉴宋代山水画与古典园林布局
- **墨韵色彩体系**：月白、黛青、竹青、赭石等传统色系
- **优雅动效**：水墨晕染、竹叶飘落、画卷展开等传统文化元素
- **书法字体系统**：精心搭配的中西文字体，呈现书卷气息

### 🎋 独特社区体验
- **雅集而非群聊**：深度讨论替代碎片化交流
- **批注而非点赞**：质量优先的价值评价体系
- **年轮而非等级**：可视化成长记录，关注长期发展
- **园林而非广场**：宁静有序的文化空间设计

### 📚 文化功能模块
- **杏坛广场**：学术讨论与思想交流
- **兰亭雅集**：主题沙龙与深度对话
- **藏书楼**：经典文献与优质内容库
- **静心斋**：冥想空间与文化熏陶
- **墨池**：创作打卡与技能练习

## 🛠 技术架构

### 基础平台
- **核心平台**：Discourse（现代化的开源社区平台）
- **部署方式**：Docker容器化部署
- **扩展架构**：插件系统 + 主题系统

### 定制开发内容
- **东方美学主题**：基于Discourse主题系统深度定制的东方美学界面
- **文化功能插件**：年轮成长系统、墨池打卡系统、雅集语音系统等
- **文化化交互**：重新设计的术语体系、交互方式和社区规则

### 特色技术实现
- **水墨动画引擎**：基于Canvas的东方美学动效
- **响应式园林布局**：多断点自适应设计系统
- **年轮可视化系统**：个人成长历程图表
- **雅集语音系统**：支持小规模高质量语音交流

## 🎨 设计资源

本项目拥有完整的视觉体系方案，包括：

- 五正色十二间色色彩系统
- 汉字意象与白描线形图标库
- 园林式与册页式布局模式
- 水墨动效与微交互规范
- 深色模式与季节主题系统

## 🌱 项目愿景

菩提树不仅是一个社交平台，更是：

1. **数字时代的书院**：复兴传统教育精神
2. **东方美学的实践**：探索数字产品中的文化表达
3. **深度交流的庇护所**：在喧嚣网络中提供一片静土
4. **成长记录的博物馆**：珍视每一段思想历程
